### Java的命名的规范

#### 规范

+ 名字必须要由数字、字母、下划线、美元符号组层，开头必须是字母或者是下划线，严格区分大小写

+ 开始必须是由字母或者是下划线
+ 严格区分大小写
+ 命名不能是关键字

#### 规则

+ 小驼峰（方法和变量）
  + 每个单词除了首字母是小写，其余的是大写（firstName）
  + 针对于方法和变量
+ 大驼峰 （类）
  + 每个单词的首字母大写（FirstName）
  + 针对于类
